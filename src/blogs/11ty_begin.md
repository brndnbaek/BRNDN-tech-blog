## Chrome Dev Summit 2019 사이트를 11ty로 만들었다고?

새로만들 블로그를 어떤 생성기로 사용할지 고민중일때 유트브의 알 수 없는 알고리즘이 11ty를 소개해주더군요. 관련 동영상이 꽤 있어서 최근 인기 있는 생성기 인가보다 하고 관심있게 찾아보게 되었습니다.

일단, 구글 데브 서밋 사이트를 11ty를 사용해서 만들었다는 점에서 좀 더 찾아보게 되었는데요. 만들어진지 얼마 않된 생성기라서인지 아니면 제작하는분들이 초보들한테 관심이 덜해서인지 정적생성기에 지식이 없는 초보가 따라하기에는 자료가 좀 부족한 느낌입니다. 

```
npm install -g @11ty/eleventy
echo '# Page header' > README.md
eleventy
```
